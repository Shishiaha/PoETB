# PoETB 1.3 - PoE 1&2 Live button clicker
PoE Trade Button, Direct live search clicker for PoE 1 &amp; 2.


This extension is provided to you freely. It enables clicking Direct Whisper & Teleport to hideout buttons only from live search directly in your tabs. It is only supported and tested with Chrome & Brave but should work on any chromium based browser. This is work in progress.

Settings :

- Enable On/Off
- Click Delay (ms)
- Teleport Pausing (ms) : time for the extension to pause when you teleported to someone's hideout.

It may contain a lot of bugs and issues. It is a very early release.

DISCLAIMER : I have no idea if it goes against the GGG Terms of usage, i have no idea if you can be banned for this tool. It does not operate server side actions, game actions, or multiple actions at once. It only works inside the browser as a simple extension. Use it at your own risks.
